Multiple Set Pauper Sealed Pool Generator

You can use these bookmarklets to generate sealed pools using commons from multiple sets across multiple blocks.

  • Setup
    • Drag each bookmarklet into your bookmarks bar.
  • Use
    • Browse to a MTG Gen page.
    • Click the MTGEN Pauper Sealed bookmarklet in your bookmarks bar.
    • Enter your set abbreviations and click OK.
    • Copy and paste the data from the new prompt into a .txt file.
    • Import the text file into MTGO.
  • Optional
    • If you're going to generate several pools for the same sets, use the Presets bookmarklet.

Bookmarklets

  • MTGEN Pauper Sealed
  • MTGEN Presets
